### Option 1: Direct Upload (Recommended)
1. **Access your hosting control panel** (cPanel, Plesk, etc.)
2. **Navigate to File Manager** or use FTP
3. **Upload these files** to your domain's root directory:
   - `index.html`

### Option 2: Using FTP Client
1. **Download an FTP client** (FileZilla, WinSCP)
2. **Connect to your hosting server** using your FTP credentials
3. **Upload the files** to your domain's public_html folder
4. **Ensure index.html is in the root directory**

### Adding Products
Edit the `products` array in `script.js`:
```javascript

    id: 13,
    name: "New Product Name",
```

### Changing Colors
Edit the color variables in `style.css`:
```css
/* Main brand color */
--primary-color: #ff6b9d;

/* Gradient colors */
--gradient-start: #667eea;
--gradient-end: #764ba2;
```
