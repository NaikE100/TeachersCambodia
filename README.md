TeachersCambodia | International Recruitment Platform

🚀 Overview

TeachersCambodia is a web-based talent sourcing platform designed to bridge the gap between international educators and the Southeast Asian academic market. Built with a focus on high-performance and responsive UI, the platform streamlines the recruitment process for both schools and applicants.
Note: The live production domain is currently undergoing an infrastructure migration to a Progressive Web App (PWA) model. The source code below represents the stable React build.

🛠 Technical Stack & Modern Workflow

This project was developed using a Modern AI-Assisted Workflow, leveraging cutting-edge tools to ensure rapid iteration and clean code architecture.
Frontend: React.js
Styling: Custom CSS3 (Flexbox/Grid) with a Mobile-First approach.
Development Tools: * Cursor IDE: Utilized for AI-assisted code optimization and debugging.
Figma: Used for UI/UX prototyping and wireframing.
Git: Version control and repository management.

✨ Key Features

Responsive UI/UX: A fully fluid design that ensures teachers can apply easily from mobile devices or desktops.
Dynamic Talent Sourcing: Custom-built logic to categorize and filter international candidate profiles.
Modern Interaction Patterns: Optimized form handling and state management for a frictionless user experience.
